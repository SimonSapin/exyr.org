Page breaks

h1, h2 { page-break-after: avoid }
pre { page-break-inside: avoid }
.slide { page-break-after: always }

Page layout

@page {
    size: A4 portrait;
    size: 210mm 297mm;
    margin: 2cm;
}
@page :left { margin-right: 4cm }
@page :right { margin-left: 4cm }

Named pages

table {
    page: landscape-table;
}
@page landscape-table {
    size: A4 landscape;
    size: 297mm 210mm;
}

page-margin boxes
